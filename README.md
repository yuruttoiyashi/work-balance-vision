# Work Balance Vision

## デモURL

https://work-balance-vision.vercel.app

## 概要

Work Balance Vision は、従業員の日々の体調・気分・集中度・業務進捗を見える化し、管理者が早期フォローにつなげるための就労コンディション管理ダッシュボードです。

障害者雇用や就労支援、社内の業務改善を想定し、体調不良・進捗遅れ・支援依頼などを自動でアラート表示できるようにしています。

公開デモ版では、採用担当者がログインなしで自由に操作できるよう、データは各ブラウザの localStorage に保存されます。  
そのため、編集・削除・CSV取込を試しても、他の閲覧者のデータや共有データは壊れません。

---

## 制作目的

このアプリは、以下のような実務課題を想定して制作しました。

- 従業員の体調や困りごとを管理者が早めに把握したい
- 日々の状態を記録し、面談や業務調整の材料にしたい
- 体調不良や進捗遅れを自動で検知したい
- CSVデータを取り込み、既存の日報や記録を活用したい
- 障害者雇用・就労支援・人事労務の現場で使える業務改善アプリを想定したい

## 主な機能

### ダッシュボード

- 本日の入力率
- 支援が必要な可能性のある人数
- 体調不良者数
- 進捗遅れ件数
- 本日の最新コンディション一覧
- 優先フォロー対象の表示

### コンディション入力

- 対象メンバー選択
- 日付入力
- 体調、気分、集中度、作業進捗の入力
- 今日の作業内容
- 困っていること
- 管理者に伝えたいこと・支援依頼

### CSV取込

- サンプルCSVダウンロード
- CSVファイル選択
- ヘッダー・入力値チェック
- 取り込み前プレビュー
- 有効データのみ取込
- localStorageへの保存

### CSV出力

- 現在の入力履歴をCSVとして出力
- 面談記録・支援記録・業務報告用データとして活用可能

### 入力履歴

- 手入力データ・CSV取込データの一覧表示
- 編集
- 削除
- 登録元の表示

### アラート

以下のような条件を自動検知します。

- 体調が「悪い」
- 作業進捗が「遅れ」
- 支援依頼がある
- 困りごとの記入がある
- 体調不良や集中度低下が連続している

## デモ版の仕様

公開デモ版では、採用担当者が自由に操作できるよう、保存先を localStorage にしています。

| 項目 | 内容 |
|---|---|
| データ保存 | 各ブラウザの localStorage |
| ログイン | なし |
| 編集・削除 | 可能 |
| CSV取込 | 可能 |
| CSV出力 | 可能 |
| 他ユーザーへの影響 | なし |
